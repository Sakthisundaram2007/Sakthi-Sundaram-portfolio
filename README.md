 Sakthi-Sundaram-portfolio
This is a responsive personal portfolio website built using HTML and Tailwind CSS. It features a clean design with sections like About, Achievements, Education, Skills, Projects, and Contact. The site includes a circular profile image with a glow effect and smooth fade-in animations on scroll, providing a modern and professional user experience.
 🌐 Personal Portfolio Website

📌 Overview
This is a modern and responsive personal portfolio website built using **HTML** and **Tailwind CSS**. It is designed to showcase personal details, skills, education, achievements, and projects in a clean and professional layout.

---

 ✨ Features

- ✅ Responsive design (mobile-friendly)
- ✅ Fixed navigation bar
- ✅ Smooth scrolling between sections
- ✅ Fade-in animation on scroll
- ✅ Circular profile image with glow effect
- ✅ Clean and minimal UI design
- ✅ Sections included:
  - About
  - Achievements
  - Education
  - Skill Sets
  - Projects
  - Contact

---

 🛠️ Technologies Used

- HTML5
- Tailwind CSS (CDN)
- JavaScript (for scroll animations)
